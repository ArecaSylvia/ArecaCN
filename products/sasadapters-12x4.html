6 Gb/s SAS RAID Controllers

ARC-12x4 internal PCIe 2.0 host RAID controllers are a cost-effective solutions for connecting up to 4/8 6Gb/s SAS peripheral devices. The RAID controllers are based on the same RAID kernel of field-proven internal/external RAID controller and same device driver architecture with widely used 3Gb/s and 6Gb/s SAS/SATA RAID controller. Applications that benefit most features from these controllers include NAS, server RAID solutions, web servers, near-line backup, security systems and streaming applications. ARC-1214-4I/1224-8I support directly attached 4/8 internal 6Gb/s SAS ports via 1/2 SFF-8087 connector.
