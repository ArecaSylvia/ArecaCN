8/16 Ports PCIe Gen 4.0 Tri-Mode RAID Adapters

The new fourth generation ARC-1886 family offers advanced technology for increased performance and improved enterprise data protection. They were designed with 1.6 GHz dual core ROC processor, DDR4-2666 memory and PCIe 4.0 interface for the most performance-hungry database and IT applications. Based on the ARM A15 PCIe 4.0 x8 to SAS/SATA/PCIe RAID on Chip (ROC) controller, the ARC-1886 series RAID adapters can offer both PCIe Gen 4.0 host and 12Gb/s SAS/SATA/PCIe (NVMe) tri-mode storage interfaces.

    RAID Features

  • RAID Levels - 0, 1, 3, 5, 6, 10, 30, 50, 60, Single Disk or JBOD
  • Multiple RAID 0 and RAID 10(1E) support (RAID 00 and RAID100)
  • Support RAID 1 - Multi Mirroring and Mirrored Pass-Through Disk
  • Multiple RAID selection
  • Configurable stripe size up to 1MB
  • Write-through or write-back cache support
  • Support read/write cache allocation by policy
  • Support HDD firmware update
  • Online array roaming
  • Online RAID level/stripe size migration
  • Online capacity expansion and RAID level migration simultaneously
  • Instant availability and background initialization
  • Support global and dedicated hot spare
  • Automatic drive insertion/removal detection and rebuilding
  • Support for native 4K and 512 byte sector SAS and SATA devices
  • Support UNMAP command
  • S.M.A.R.T. support
  • Multiple pairs SSD/HDD disk clone function
  • SSD automatic monitor clone (AMC)
  • Controller based hardware encryption function
  • SED (self-encryption drives) function support
  • Support backplanes based on the SFF-TA-1005 specification (UBM)
  • Redundant flash image for adapter availability
  • Support intelligent power management to save energy and extend service life
  • Support NTP protocol synchronize RAID adapter clock over the onboard Ethernet port

    SFF-8654[x8] To 2X SFF-8611[x4]
    Use for backplanes with x4 SFF-8612 connectors with pinouts that follow the SFF-9402 specification (Intel/Broadcom VPP).


    SFF-8654[x8] To 4X SFF-8611[x2]
    Use for backplanes with x4 SFF-8612 connectors with pinouts that follow the SFF-9402 specification (intel/Broadcom VPP).
    For U2 NVMe drives requiring a common REFCLK, one REFCLK is supplied per SFF-8612 OCulink connector. For x4 NVMe connections, the REFCLK sourced by each connector directly clocks each attached x4 PCIe connection. To directly attach x2/x1 NVMe drives that require a common REFCLK, where more than one NVMe drive (2/4) is sourced from a single SFF-8654 connector on the ARC-1886 RAID adapter, take care to properly fan-out the shared REFCLK on the backplane design.


    SFF-8654[x8] To 8X SFF-8611[x1]
    Use for backplanes with x8 SFF-8612 connectors with pinouts that follow the SFF-9402 specification (intel VPP).
    For U2 NVMe drives requiring a common REFCLK, one REFCLK is supplied per SFF-8612 OCulink connector. For x4 NVMe connections, the REFCLK sourced by each connector directly clocks each attached x4 PCIe connection. To directly attach x2/x1 NVMe drives that require a common REFCLK, where more than one NVMe drive (2/4) is sourced from a single SFF-8654 connector on the ARC-1886 RAID adapter, take care to properly fan-out the shared REFCLK on the backplane design.


    SFF-8654[x8] To 8X SATA[x1]
    Use for traditional 6Gb/s SAS/SATA connections. Usually backplanes designed to support SAS/SATA only and SAA/SATA drivers use SATA connectors.


    SFF-8654[x8] To 8X U3 SFF-8639[x1]
    Enables direct connect from the adapter to a U.3 NVMe (x1) or SAS/SATA drive.


    SFF-8654[x8] To 2X SFF-8643[x4]
    Use for traditional SAS/SATA connections. Usually backplanes designed to support SAS/SATA only and SAS/SATA drives use SFF-8643 connectors (SGPIO).


    SFF-8654[x8] To 1X SFF-8654[x8]
    Use for backplanes with x8 SFF-8654 connectors with pinouts that follow the SFF-9402 specification.


    SFF-8654[x8] To 2X SFF-8654[x4]
    Use for backplanes with x8 SFF-8654 connectors with pinouts that follow the SFF-9402 specification.


    SFF-8654[x8] To 2X SFF-8087[x4]
